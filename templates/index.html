About

This challenge was made in approximately 2-3 weeks and is still kinda in development. As a result, there may be some bugs, so please bear with me and contact me if you find one. I will try to add new puzzles weekly, but I am unable to guarentee this.
There are currently 16 puzzles in circulation, but you will get assigned 4 random ones. The entire backend of this is written in Python and if you are still curious, you can go the the GitHub page where the code is hosted.
